üéì Theory: Events & Methods

In Vue, you handle events using the v-on directive (or @ shorthand) and define methods in the methods option.

Event Handling Syntax:

<button v-on:click="methodName">Click me</button>
<!-- Shorthand -->
<button @click="methodName">Click me</button>

Passing Arguments:

<button @click="sayHello('John')">Say Hello</button>

Accessing Event Object:

<button @click="handleClick">Click</button>
// In methods:
handleClick(event) {
    console.log(event.target);
}

Event Modifiers:

  • .stop - Stop event propagation
  • .prevent - Prevent default behavior
  • .once - Trigger only once
  • .enter - Only trigger on Enter key
