üéÆ Building a Number Guessing Game - Step by Step Tutorial

Let's build a number guessing game! This will teach you event handling, methods, and form handling in Vue.

Step 1: Set Up the Data Structure

What we're doing: Define all the data our game needs to track.

Key concept: We need to store the secret number, user's guess, feedback message, and guess count.

data() { return { secretNumber: 0, // The number to guess userGuess: '', // User's input message: '', // Feedback message guessCount: 0, // Number of guesses gameStarted: false // Track if game is active } }

Explanation:

  • secretNumber - The random number the user needs to guess (1-100)
  • userGuess - What the user types in the input field
  • message - Feedback like "Too high", "Too low", or "Correct!"
  • guessCount - How many times the user has guessed
  • gameStarted - Whether a game is currently active

Step 2: Generate Random Number Method

What we're doing: Create a method that generates a random number between 1 and 100.

Key concept: Methods are functions defined in the methods option.

methods: { startNewGame() { // Generate random number between 1 and 100 this.secretNumber = Math.floor(Math.random() * 100) + 1; // Reset game state this.userGuess = ''; this.message = ''; this.guessCount = 0; this.gameStarted = true; // For testing (remove in production) console.log('Secret number:', this.secretNumber); } }

Explanation:

  • Math.random() - Returns a number between 0 and 1
  • Math.random() * 100 - Gives us 0 to 99.999...
  • Math.floor(...) - Rounds down to whole number (0-99)
  • + 1 - Shifts range to 1-100
  • We reset all game state when starting a new game
üí° Tip: We use console.log to see the secret number for testing. In a real game, you'd remove this!

Step 3: Create the Guess Method

What we're doing: Create a method that checks the user's guess and provides feedback.

Key concept: Methods can access data using this and update it.

methods: { makeGuess() { // Convert userGuess to a number const guess = Number(this.userGuess); // Increment guess count this.guessCount++; // Check the guess and provide feedback if (guess === this.secretNumber) { this.message = 'Correct! üéâ'; } else if (guess > this.secretNumber) { this.message = 'Too high! ‚¨áÔ∏è'; } else { this.message = 'Too low! ‚¨ÜÔ∏è'; } // Clear the input field this.userGuess = ''; } }

Explanation:

  • Number(this.userGuess) - Converts string input to number
  • this.guessCount++ - Increments the guess counter
  • We use if/else if/else to check the guess
  • Update this.message with appropriate feedback
  • Clear the input field after each guess

Step 4: Add the HTML Structure

What we're doing: Create the UI with buttons and input field.

Key concept: Use @click for button clicks and @submit.prevent for form submission.

<div id="app"> <h2>Number Guessing Game</h2> <p>I'm thinking of a number between 1 and 100. Can you guess it?</p> <button @click="startNewGame"> New Game </button> <form @submit.prevent="makeGuess" v-if="gameStarted"> <input type="number" v-model.number="userGuess" placeholder="Enter your guess" min="1" max="100"> <button type="submit"> Guess </button> </form> <div v-if="message" class="message"> {{ message }} </div> <p v-if="guessCount > 0"> Guesses: {{ guessCount }} </p> </div>

Explanation:

  • @click="startNewGame" - Calls method when button is clicked
  • @submit.prevent="makeGuess" - Prevents form submission and calls method
  • .prevent - Event modifier that prevents default form behavior
  • v-model.number - Automatically converts input to number
  • v-if="gameStarted" - Only show form when game is active
  • v-if="message" - Only show message when it exists

Step 5: Add Dynamic Styling Based on Message

What we're doing: Use :class to style the message differently based on the feedback.

<div v-if="message" class="message" :class="{ 'correct': message.includes('Correct'), 'too-high': message.includes('high'), 'too-low': message.includes('low') }"> {{ message }} </div>

Explanation:

  • :class - Dynamically adds CSS classes
  • message.includes('Correct') - Checks if message contains "Correct"
  • Different classes are applied based on the message content
  • This gives visual feedback (green for correct, red for too high, yellow for too low)

const { createApp } = Vue; createApp({ data() { return { secretNumber: 0, userGuess: '', message: '', guessCount: 0, gameStarted: false } }, methods: { startNewGame() { // Generate random number between 1 and 100 this.secretNumber = Math.floor(Math.random() * 100) + 1; // Reset game state this.userGuess = ''; this.message = ''; this.guessCount = 0; this.gameStarted = true; // For testing (remove in production) console.log('Secret number:', this.secretNumber); }, makeGuess() { // Convert input to number const guess = Number(this.userGuess); // Validate input if (isNaN(guess) || guess < 1 || guess > 100) { this.message = 'Please enter a number between 1 and 100!'; return; } // Increment guess count this.guessCount++; // Check the guess and provide feedback if (guess === this.secretNumber) { this.message = 'Correct! üéâ'; } else if (guess > this.secretNumber) { this.message = 'Too high! ‚¨áÔ∏è'; } else { this.message = 'Too low! ‚¨ÜÔ∏è'; } // Clear input field this.userGuess = ''; } } }).mount('#app');

üéì Key Concepts You Learned

  1. @click: Handle button click events
  2. @submit.prevent: Handle form submission and prevent default behavior
  3. methods: Define functions that can be called from the template
  4. this: Access data and other methods within methods
  5. Math.random(): Generate random numbers
  6. Number(): Convert strings to numbers
  7. v-model.number: Two-way binding with automatic number conversion
  8. :class: Dynamic CSS classes based on conditions
  9. Event modifiers: .prevent, .enter, etc. to modify event behavior
