üéì Theory: Template Syntax

Vue uses a template syntax that extends HTML. The most important concept is interpolation - displaying data in your HTML.

1. Text Interpolation (Mustache Syntax)

<span>{{ message }}</span>

This displays the value of message from your data.

2. Raw HTML

<div v-html="rawHtml"></div>

Use v-html to render HTML (be careful with user input!).

3. Attribute Binding

<div v-bind:id="dynamicId"></div>
<!-- Shorthand -->
<div :id="dynamicId"></div>

Bind attributes dynamically using v-bind or : shorthand.

4. JavaScript Expressions

{{ number + 1 }}
{{ ok ? 'YES' : 'NO' }}
{{ message.split('').reverse().join('') }}

You can use JavaScript expressions inside {{ }}.
