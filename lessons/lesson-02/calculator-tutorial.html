Let's build a calculator together! I'll explain each step so you understand how Vue works.

Step 1: Set Up the Basic Structure

What we're doing: First, we need to create a Vue app and set up the HTML structure.

Key concept: We'll use v-model to create two-way data binding with input fields.

<div id="app"> <h2>Simple Calculator</h2> <input type="number" v-model="number1" placeholder="First number"> <input type="number" v-model="number2" placeholder="Second number"> </div>

Explanation:

  • v-model="number1" creates a two-way binding - when you type, it updates the data, and when data changes, the input updates
  • type="number" ensures only numbers can be entered

Step 2: Add Data Properties

What we're doing: Define the data that our calculator needs.

data() { return { number1: 0, // First number from input number2: 0 // Second number from input } }

Explanation:

  • number1 and number2 store the values from the input fields
  • They start at 0, but will update automatically when the user types (thanks to v-model!)

Step 3: Display Calculations Using JavaScript Expressions

What we're doing: Use Vue's template syntax to perform calculations directly in the HTML.

Key concept: You can write JavaScript expressions inside {{ }}!

<div class="result"> <p>Sum: {{ number1 + number2 }}</p> <p>Difference: {{ number1 - number2 }}</p> <p>Product: {{ number1 * number2 }}</p> <p>Quotient: {{ number1 / number2 }}</p> </div>

Explanation:

  • {{ number1 + number2 }} - Vue automatically calculates this whenever number1 or number2 changes!
  • This is called reactivity - the UI updates automatically when data changes
  • No need to manually update the DOM - Vue does it for you!

Step 4: Handle Division by Zero

What we're doing: Use a conditional expression to show an error when dividing by zero.

Key concept: Ternary operator condition ? valueIfTrue : valueIfFalse

<p>Quotient: {{ number2 !== 0 ? (number1 / number2).toFixed(2) : 'Cannot divide by zero!' }} </p>

Explanation:

  • number2 !== 0 checks if the second number is NOT zero
  • If true: calculate and show the division result (with 2 decimal places using toFixed(2))
  • If false: show the error message
  • This is a ternary operator - a shorthand if/else

Here's the complete JavaScript code:

üéì Key Concepts You Learned

  1. v-model: Two-way data binding - connects input fields to data
  2. v-model.number: Automatically converts input to number type
  3. {{ }}: Interpolation - displays data and executes JavaScript expressions
  4. Reactivity: Vue automatically updates the UI when data changes
  5. Ternary Operator: condition ? valueIfTrue : valueIfFalse
  6. v-if / v-else: Conditionally show/hide elements
