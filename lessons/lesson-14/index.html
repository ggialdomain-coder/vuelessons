What is Dynamic Styling?

Vue allows you to dynamically apply CSS classes and inline styles based on your component's data. This enables responsive, interactive styling.

Class Binding with :class

Use :class (or v-bind:class) to dynamically bind CSS classes:

Object Syntax:

// Toggle class based on condition
<div :class="{ active: isActive, error: hasError }">
    Content
</div>

// In data
data() {
    return {
        isActive: true,
        hasError: false
    }
}

Array Syntax:

// Apply multiple classes
<div :class="[activeClass, errorClass]">
    Content
</div>

// In data
data() {
    return {
        activeClass: 'active',
        errorClass: 'error'
    }
}

Style Binding with :style

Use :style (or v-bind:style) to dynamically bind inline styles:

Object Syntax:

// Apply inline styles
<div :style="{ color: textColor, fontSize: fontSize + 'px' }">
    Content
</div>

// In data
data() {
    return {
        textColor: 'red',
        fontSize: 20
    }
}

Array Syntax:

// Apply multiple style objects
<div :style="[baseStyles, dynamicStyles]">
    Content
</div>

Use Cases:

  • ‚úÖ Theme switching (light/dark mode)
  • ‚úÖ Status indicators (success/error/warning)
  • ‚úÖ Interactive UI elements (hover, active states)
  • ‚úÖ Progress bars and dynamic sizing
  • ‚úÖ Conditional styling based on data

Best Practices:

  • ‚úÖ Use :class for CSS classes (better performance)
  • ‚úÖ Use :style for dynamic values (colors, sizes)
  • ‚úÖ Prefer computed properties for complex style logic
  • ‚úÖ Keep style objects in data or computed properties
