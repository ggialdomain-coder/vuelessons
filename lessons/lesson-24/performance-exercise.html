Your Task: Optimize a component with performance issues.

Part 1: Replace v-if with v-show

  1. Identify elements that toggle frequently
  2. Replace v-if with v-show for frequent toggles
  3. Keep v-if for elements that rarely toggle

Part 2: Use Computed Instead of Methods

  1. Find methods called in templates
  2. Convert to computed properties if they derive from reactive data
  3. Test that results are the same but performance is better

Part 3: Add v-memo to Lists

  1. Identify expensive list items
  2. Add v-memo with appropriate dependencies
  3. Test that items only re-render when dependencies change

üí° Hints

  • Use browser DevTools Performance tab to measure
  • v-show for dropdowns, modals that toggle often
  • v-if for conditional content that rarely changes
  • Computed for values calculated from reactive data
  • v-memo for lists with expensive rendering
