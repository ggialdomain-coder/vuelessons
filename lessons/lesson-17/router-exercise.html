Your Task: Build a complete multi-page application using Vue Router.

Part 1: Setup

  1. Include Vue 3 and Vue Router 4 from CDN
  2. Create a router instance with hash history

Part 2: Create Components

  1. Home Component: Welcome message and brief intro
  2. Products Component: Display a list of at least 3 products
  3. About Component: Information about your app/company
  4. Contact Component: Contact form or contact information

Part 3: Define Routes

  1. Route '/' ‚Üí Home component
  2. Route '/products' ‚Üí Products component
  3. Route '/about' ‚Üí About component
  4. Route '/contact' ‚Üí Contact component

Part 4: Navigation

  1. Create a navigation menu with router-link components
  2. Add router-view to display current page
  3. Style the navigation (active link highlighting)

Part 5: Bonus Features

  1. Add a "Back" button that uses $router.go(-1)
  2. Show current route path in each component
  3. Add a 404/NotFound component for unknown routes

üí° Hints

  • Use createWebHashHistory() for hash-based routing
  • Remember to use app.use(router) to register the router
  • Use <router-link to="/path"> for navigation
  • Use <router-view></router-view> to display route components
  • Access current route with this.$route.path
