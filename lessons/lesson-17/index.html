What is Vue Router?

Vue Router is the official routing library for Vue.js. It allows you to build Single Page Applications (SPAs) with multiple "pages" without full page reloads.

Why Use Vue Router?

  • ‚úÖ Create multi-page apps without page reloads
  • ‚úÖ Browser back/forward buttons work
  • ‚úÖ Shareable URLs for different pages
  • ‚úÖ Better user experience

Basic Setup:

// Include Vue Router from CDN
<script src="https://unpkg.com/vue-router@4/dist/vue-router.global.js"></script>

// Define routes
const routes = [
    { path: '/', component: Home },
    { path: '/about', component: About },
    { path: '/contact', component: Contact }
];

// Create router
const router = VueRouter.createRouter({
    history: VueRouter.createWebHashHistory(),
    routes
});

// Use router in app
const app = createApp({});
app.use(router);
app.mount('#app');

Router Components:

  • <router-view> - Displays the current route component
  • <router-link> - Navigation links (like <a> but for routes)

Navigation:

// In template
<router-link to="/about">About</router-link>

// In JavaScript
this.$router.push('/about');
router.push('/about');

Key Concepts:

  • Routes: URL paths mapped to components
  • Router View: Where route components are displayed
  • Router Link: Navigation links that don't reload the page
  • History Mode: How URLs are handled (hash vs HTML5)
