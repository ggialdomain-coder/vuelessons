üìñ Building a Multi-Page App with Vue Router

Goal: Build a complete multi-page application with Home, Products, About, and Contact pages using Vue Router.

Step 1: Include Vue Router

First, include both Vue 3 and Vue Router 4 from CDN in your HTML.

<!-- Include Vue 3 --> <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script> <!-- Include Vue Router 4 --> <script src="https://unpkg.com/vue-router@4/dist/vue-router.global.js"></script>
üí° Note: Make sure Vue Router is loaded after Vue, as it depends on Vue.

Step 6: Define Routes

Map URL paths to your components. Each route connects a path to a component.

const routes = [ { path: '/', component: Home }, { path: '/products', component: Products }, { path: '/about', component: About }, { path: '/contact', component: Contact } ];
üí° Key Points:
  • path: '/' - The home/root route
  • Each path maps to a component
  • Paths are case-sensitive

Step 7: Create the Router

Create the router instance with hash history mode (works without server configuration).

const { createRouter, createWebHashHistory } = VueRouter; const router = createRouter({ history: createWebHashHistory(), // Uses # in URL (e.g., /#/products) routes });
üí° Hash History: URLs will look like /#/products instead of /products. This works everywhere without server configuration, perfect for learning!

Step 8: Create and Configure the App

Create the Vue app and register the router with it.

const { createApp } = Vue; const app = createApp({}); app.use(router); // Register the router app.mount('#app'); // Mount to the #app element
üí° Important: You must call app.use(router) before app.mount()!

Step 9: Add Navigation Menu

Create a navigation menu using <router-link> components. These create navigation links that don't reload the page.

<div id="app"> <nav> <router-link to="/">Home</router-link> <router-link to="/products">Products</router-link> <router-link to="/about">About</router-link> <router-link to="/contact">Contact</router-link> </nav> <!-- Route components display here --> <router-view></router-view> </div>
üí° Key Components:
  • <router-link to="/path"> - Creates a navigation link
  • <router-view></router-view> - Displays the current route component
  • Active links automatically get the router-link-active class for styling

Step 10: Style the Navigation

Add CSS to style your navigation menu and highlight active links.

nav { background: #42b983; padding: 15px; border-radius: 5px; margin: 10px 0; } nav a { color: white; text-decoration: none; margin: 0 15px; padding: 8px 15px; border-radius: 3px; } nav a:hover, nav a.router-link-active { background: rgba(255,255,255,0.2); }
üí° Active Link: Vue Router automatically adds the router-link-active class to the active link, so you can style it differently!

Step 12: Bonus - Add a Back Button

Add a "Back" button that uses $router.go(-1) to go back in browser history.

// In a component methods: { goBack() { this.$router.go(-1); // Go back one page in history } }
<!-- In template --> <button @click="goBack">Back</button>

Step 14: Bonus - Add 404/NotFound Component

Create a NotFound component for unknown routes and add it as a catch-all route.

const NotFound = { template: '
' + '

404 - Page Not Found

' + '

The page you are looking for does not exist.

' + 'Go Home' + '
' }; // Add as last route (catch-all) const routes = [ { path: '/', component: Home }, { path: '/products', component: Products }, { path: '/about', component: About }, { path: '/contact', component: Contact }, { path: '/:pathMatch(.*)*', component: NotFound } // Catch-all route ];
üí° Catch-All Route: The path /:pathMatch(.*)* matches any route that doesn't match the previous routes. Always put it last!

Create components const Home = { template: '...' }; const Products = { template: '...' }; const About = { template: '...' }; const Contact = { template: '...' }; // 3. Define routes const routes = [ { path: '/', component: Home }, { path: '/products', component: Products }, { path: '/about', component: About }, { path: '/contact', component: Contact } ]; // 4. üéì Key Concepts You Learned

  1. Vue Router: Official routing library for creating multi-page apps
  2. Components: Each "page" is a Vue component
  3. Routes: Map URL paths to components
  4. router-link: Navigation links that don't reload the page
  5. router-view: Displays the current route component
  6. Hash History: Uses # in URLs (works without server config)
  7. $route: Access current route information
  8. $router: Programmatic navigation methods
