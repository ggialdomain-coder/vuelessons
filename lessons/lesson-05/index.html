üéì Theory: Computed vs Methods vs Watchers

Computed Properties:

  • Cached: Only recalculates when dependencies change
  • Use for: Derived data that depends on other data
  • Syntax: computed: { propertyName() { return ... } }

Methods:

  • Not cached: Runs every time it's called
  • Use for: Actions, event handlers

Watchers:

  • Reactive: Runs when watched property changes
  • Use for: Side effects, async operations, validation
  • Syntax: watch: { propertyName(newVal, oldVal) { ... } }

When to Use What:

  • Computed: Displaying filtered/sorted lists, calculations
  • Methods: Event handlers, actions
  • Watchers: API calls, logging, complex side effects
