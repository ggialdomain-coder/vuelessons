üéì Theory: Deployment & Build Tools

Why Build Tools?

Build tools optimize your Vue app for production by bundling, minifying, and optimizing your code.

Popular Build Tools:

  • Vite: Fast, modern build tool (recommended for Vue 3)
  • Webpack: Powerful, flexible bundler
  • Vue CLI: Official Vue.js tooling

What Build Tools Do:

  • Bundling: Combine multiple files into optimized bundles
  • Minification: Reduce file size by removing whitespace
  • Code Splitting: Split code into smaller chunks for faster loading
  • Tree Shaking: Remove unused code
  • Transpilation: Convert modern JavaScript to compatible versions

Deployment Options:

  • Netlify: Easy deployment with continuous integration
  • Vercel: Great for static sites and serverless functions
  • GitHub Pages: Free hosting for static sites
  • Cloudflare Pages: Fast global CDN
  • Traditional Hosting: Upload built files to any web server

Basic Vite Setup:

npm create vue@latest my-app
cd my-app
npm install
npm run dev      # Development server
npm run build    # Build for production
npm run preview  # Preview production build

Build Output:

After building, you'll have optimized files in a dist folder ready for deployment.
