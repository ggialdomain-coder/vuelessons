Your Task: Build a shopping cart system using state management where multiple components share the same cart state.

Part 1: Create the Store

  1. Create a cartStore object with:
    • State: items array and total number
    • Method: addItem(product) - adds product to cart
    • Method: removeItem(itemId) - removes item from cart
    • Method: clearCart() - empties the cart
    • Method: calculateTotal() - updates total price
  2. Use Vue.reactive() for the state

Part 2: Create ProductList Component

  1. Display a list of products (at least 3 products)
  2. Each product should show: name, price, and "Add to Cart" button
  3. When "Add to Cart" is clicked, add the product to the store's cart

Part 3: Create CartDisplay Component

  1. Display all items in the cart from the store
  2. Show item name, price, and "Remove" button for each item
  3. Display the total price
  4. Show "Clear Cart" button
  5. Show message when cart is empty

Part 4: Test the Integration

  1. Add items from ProductList - they should appear in CartDisplay
  2. Remove items from CartDisplay - they should disappear
  3. Total should update automatically
  4. Clear cart should empty everything

üí° Hints

  • Use Vue.reactive() to make the store state reactive
  • Access store state in components via data() { return { store: cartStore.state } }
  • Call store methods from component methods: cartStore.addItem(product)
  • Use v-for to display lists
  • Calculate total by summing all item prices
