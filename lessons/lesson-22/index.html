What are Composables?

Composables are reusable functions that contain reactive state and logic. They're like custom hooks that can be shared across components.

Why Use Composables?

  • ‚úÖ Reuse logic across multiple components
  • ‚úÖ Better code organization
  • ‚úÖ Easier to test
  • ‚úÖ Share stateful logic

Creating a Composable:

// useCounter.js - A reusable composable
function useCounter(initialValue = 0) {
    const count = ref(initialValue);
    
    const increment = () => {
        count.value++;
    };
    
    const decrement = () => {
        count.value--;
    };
    
    return {
        count,
        increment,
        decrement
    };
}

// Use in component
export default {
    setup() {
        const { count, increment, decrement } = useCounter(10);
        return { count, increment, decrement };
    }
}

Key Concepts:

  • Composable Function: A function that returns reactive state and methods
  • Naming: Usually starts with "use" (e.g., useCounter, useAuth)
  • Reusability: Can be used in multiple components
  • Stateful: Can contain reactive state
