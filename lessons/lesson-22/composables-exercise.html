Your Task: Create reusable composable functions and use them in components.

Part 1: Create useCounter Composable

  1. Create a useCounter function that:
    • Takes an initialValue parameter (default 0)
    • Returns count (ref)
    • Returns increment() method
    • Returns decrement() method
    • Returns reset() method

Part 2: Create useToggle Composable

  1. Create a useToggle function that:
    • Takes an initialValue parameter (default false)
    • Returns isOpen (ref)
    • Returns toggle() method
    • Returns open() method
    • Returns close() method

Part 3: Use Both Composables

  1. Use useCounter in a component
  2. Use useToggle in the same component
  3. Display both in the template
  4. Test that they work independently

Part 4: Use Composable Multiple Times

  1. Use useCounter twice in the same component
  2. Give them different initial values
  3. Verify they have independent state

üí° Hints

  • Composables are just functions that return reactive state and methods
  • Use ref() for reactive state
  • Return an object with all the state and methods
  • Use destructuring to get values: const { count, increment } = useCounter()
  • Rename when using multiple times: const { count: count1 } = useCounter()
