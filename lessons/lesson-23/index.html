üéì Theory: Teleport & Suspense

What is Teleport?

<Teleport> allows you to render a component's content to a different part of the DOM, outside the component's hierarchy.

Use Case: Modals, tooltips, notifications that need to be at the root level.

What is Suspense?

<Suspense> allows you to show fallback content while async components are loading.

Use Case: Loading states for async components or data fetching.

Teleport Syntax:

<Teleport to="body">
    <div class="modal">
        Modal content here
    </div>
</Teleport>

Suspense Syntax:

<Suspense>
    <template #default>
        <AsyncComponent />
    </template>
    <template #fallback>
        <div>Loading...</div>
    </template>
</Suspense>
