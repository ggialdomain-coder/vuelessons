What is Composition API?

The Composition API is an alternative way to write Vue components using functions instead of options. It provides better code organization and reusability.

Options API vs Composition API:

// Options API (what you've learned so far)
export default {
    data() {
        return { count: 0 }
    },
    methods: {
        increment() {
            this.count++
        }
    }
}

// Composition API (new way)
import { ref } from 'vue'
export default {
    setup() {
        const count = ref(0)
        const increment = () => {
            count.value++
        }
        return { count, increment }
    }
}

Key Benefits:

  • ‚úÖ Better code organization (group related logic together)
  • ‚úÖ Easier to reuse logic across components
  • ‚úÖ Better TypeScript support
  • ‚úÖ More flexible than Options API

Key Concepts:

  • setup(): Entry point for Composition API
  • ref(): Creates reactive reference for primitive values
  • reactive(): Creates reactive object
  • computed(): Creates computed properties
  • watch(): Watches for changes
