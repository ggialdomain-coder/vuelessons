v-model with Forms:

v-model creates two-way data binding with form inputs. It automatically syncs the input value with component data.

Input Types:

// Text input
<input v-model="name" type="text">

// Number input
<input v-model.number="age" type="number">

// Checkbox
<input v-model="agreed" type="checkbox">

// Radio buttons
<input v-model="gender" type="radio" value="male">
<input v-model="gender" type="radio" value="female">

// Select dropdown
<select v-model="country">
    <option value="us">USA</option>
    <option value="uk">UK</option>
</select>

// Textarea
<textarea v-model="message"></textarea>

v-model Modifiers:

  • .number - Converts value to number: v-model.number="age"
  • .trim - Removes whitespace: v-model.trim="name"
  • .lazy - Updates on change event (not input): v-model.lazy="email"

Form Submission:

<form @submit.prevent="handleSubmit">
    <input v-model="name">
    <button type="submit">Submit</button>
</form>

methods: {
    handleSubmit() {
        // Handle form submission
        console.log('Form submitted:', this.name);
    }
}

Form Validation:

  • Use computed properties to check if form is valid
  • Show error messages conditionally with v-if
  • Disable submit button when form is invalid
