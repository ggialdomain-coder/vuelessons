Step 1: Understanding What We're Building

Goal: Create a registration form with validation that collects user information and validates it before submission.

What we need:

  • Form fields: name, email, password, age, country, terms checkbox
  • v-model binding for each field
  • Form validation (check if fields are valid)
  • Error messages for invalid fields
  • Form submission handler

What is v-model?

  • ‚úÖ Two-way data binding - automatically syncs input value with component data
  • ‚úÖ Works with: text, number, email, password, checkbox, radio, select, textarea
  • ‚úÖ Much easier than manually handling input events!
üí° v-model is shorthand for: v-bind:value + @input event handler. Vue does all the work for you!

Step 2: Set Up Form Data

What we're doing: Create data properties for all form fields.

Key concept: Each form field needs a corresponding data property.

data() { return { name: '', email: '', password: '', age: null, country: '', terms: false, submitted: false } }

Explanation:

  • name, email, password - Strings (start empty)
  • age - Number (start as null)
  • country - String (start empty)
  • terms - Boolean (start as false)
  • submitted - Boolean to track if form was submitted

Step 3: Create Form Fields with v-model

What we're doing: Create form inputs and bind them with v-model.

Key concept: Use v-model to bind each input to data.

<form @submit.prevent="handleSubmit"> <div class="form-group"> <label>Name:</label> <input v-model="name" type="text" placeholder="Enter your name"> </div> <div class="form-group"> <label>Email:</label> <input v-model="email" type="email" placeholder="Enter your email"> </div> <div class="form-group"> <label>Password:</label> <input v-model="password" type="password" placeholder="Enter password"> </div> <div class="form-group"> <label>Age:</label> <input v-model.number="age" type="number" placeholder="Enter your age"> </div> <div class="form-group"> <label>Country:</label> <select v-model="country"> <option value="">Select a country</option> <option value="us">USA</option> <option value="uk">UK</option> <option value="ca">Canada</option> </select> </div> <div class="form-group"> <label> <input v-model="terms" type="checkbox"> I agree to the terms and conditions </label> </div> <button type="submit">Submit</button> </form>

Explanation:

  • v-model="name" - Binds text input to name data
  • v-model="email" - Binds email input to email data
  • v-model.number="age" - Converts age to number automatically
  • v-model="country" - Binds select dropdown to country data
  • v-model="terms" - Binds checkbox to terms boolean
  • @submit.prevent - Prevents form submission, calls handleSubmit

Step 4: Add Form Validation

What we're doing: Create a computed property to check if the form is valid.

Key concept: Use computed property to validate all fields at once.

computed: { isFormValid() { return this.name.length >= 2 && this.email.includes('@') && this.email.includes('.') && this.password.length >= 6 && this.age >= 18 && this.age <= 120 && this.country !== '' && this.terms === true; } }

Explanation:

  • name.length >= 2 - Name must be at least 2 characters
  • email.includes('@') && email.includes('.') - Basic email validation
  • password.length >= 6 - Password must be at least 6 characters
  • age >= 18 && age <= 120 - Age must be between 18-120
  • country !== '' - Country must be selected
  • terms === true - Terms must be checked
  • Returns true only if ALL validations pass

Step 5: Show Error Messages

What we're doing: Display error messages for invalid fields using v-if.

Key concept: Show errors conditionally based on validation.

<div class="form-group"> <label>Name:</label> <input v-model="name" type="text" placeholder="Enter your name"> <span v-if="name && name.length < 2" class="error"> Name must be at least 2 characters </span> </div> <div class="form-group"> <label>Email:</label> <input v-model="email" type="email" placeholder="Enter your email"> <span v-if="email && !email.includes('@')" class="error"> Please enter a valid email </span> </div> <div class="form-group"> <label>Password:</label> <input v-model="password" type="password" placeholder="Enter password"> <span v-if="password && password.length < 6" class="error"> Password must be at least 6 characters </span> </div>

Explanation:

  • v-if="name && name.length < 2" - Shows error only if name exists and is too short
  • v-if="email && !email.includes('@')" - Shows error if email exists but invalid
  • Check if field exists first (name &&) to avoid showing errors on empty fields
  • Error messages appear below each invalid field

Step 6: Handle Form Submission

What we're doing: Create the submit handler and disable button when form is invalid.

Key concept: Only submit if form is valid, show submitted data.

// In template <button type="submit" :disabled="!isFormValid">Submit</button> // In methods methods: { handleSubmit() { if (this.isFormValid) { this.submitted = true; console.log('Form submitted:', { name: this.name, email: this.email, age: this.age, country: this.country }); } } }

Explanation:

  • :disabled="!isFormValid" - Disables button when form is invalid
  • @submit.prevent="handleSubmit" - Calls handler on form submit
  • this.submitted = true - Marks form as submitted
  • Check isFormValid before processing

Step 7: Display Submitted Data

What we're doing: Show the submitted form data after successful submission.

Key concept: Use v-if to show submitted data only when form is submitted.

<div v-if="submitted" class="submitted-data"> <h3>Registration Successful!</h3> <p><strong>Name:</strong> {{ name }}</p> <p><strong>Email:</strong> {{ email }}</p> <p><strong>Age:</strong> {{ age }}</p> <p><strong>Country:</strong> {{ country }}</p> </div>

Explanation:

  • v-if="submitted" - Only shows when form is submitted
  • Displays all submitted form data
  • Shows success message to user
