üìù Registration Form Exercise

Build a registration form with validation! Use v-model to bind form inputs and validate the data.

üìã Requirements:

  1. Create form fields: name, email, password, age, country (select), and terms checkbox
  2. Use v-model to bind each input to data
  3. Add form validation:
    • Name: required, at least 2 characters
    • Email: required, valid email format
    • Password: required, at least 6 characters
    • Age: required, must be a number, between 18-120
    • Country: required
    • Terms: must be checked
  4. Show error messages for invalid fields
  5. Handle form submission with @submit.prevent
  6. Display submitted data when form is valid
  7. Disable submit button when form is invalid

üí° Hints:

  • Use v-model for all inputs: v-model="name", v-model="email", etc.
  • Use v-model.number for age: v-model.number="age"
  • Create computed property isFormValid() to check all validations
  • Show errors with v-if: <span v-if="errors.name" class="error">{{ errors.name }}</span>
  • Email validation: check if contains '@' and '.'
  • Use @submit.prevent="handleSubmit" on form
  • Disable button: <button :disabled="!isFormValid">
