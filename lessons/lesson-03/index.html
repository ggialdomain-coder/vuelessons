üéì Theory: What are Directives?

Directives are special attributes with the v- prefix that tell Vue to do something to the DOM element.

Common Directives:

  • v-if / v-else-if / v-else: Conditionally render elements
  • v-show: Toggle visibility (element always exists in DOM)
  • v-for: Render lists
  • v-bind: Bind attributes (shorthand: :)
  • v-on: Listen to events (shorthand: @)
  • v-model: Two-way data binding

v-if vs v-show:

  • v-if: Element is removed from DOM when false (better for expensive elements)
  • v-show: Element stays in DOM, just hidden with CSS (better for frequent toggles)
