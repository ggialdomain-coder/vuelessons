Let's build a todo list together! This will teach you v-for, v-if, v-model, and :class binding.

Step 1: Set Up the Data Structure

What we're doing: First, we need to define what a "todo" looks like and create an array to store todos.

Key concept: Each todo needs an id (unique identifier), text (the task), and completed (whether it's done).

data() { return { todos: [ { id: 1, text: 'Learn Vue.js', completed: false }, { id: 2, text: 'Build a todo app', completed: false } ], newTodoText: '' // For the input field } }

Explanation:

  • todos is an array of todo objects
  • Each todo has: id (unique number), text (task description), completed (true/false)
  • newTodoText will store what the user types in the input field

Step 2: Display Todos with v-for

What we're doing: Use v-for to loop through the todos array and display each one.

Key concept: v-for creates a new element for each item in an array.

<div v-for="todo in todos" :key="todo.id" class="todo-item"> {{ todo.text }} </div>

Explanation:

  • v-for="todo in todos" - loops through the todos array
  • todo is the current item in the loop
  • :key="todo.id" - Vue needs a unique key for each item (important for performance!)
  • {{ todo.text }} - displays the text of each todo
üí° Important: Always use :key with v-for! It helps Vue track which items changed, making updates faster.

Step 3: Show "No todos" Message with v-if

What we're doing: Use v-if to conditionally show a message when the list is empty.

Key concept: v-if only renders the element if the condition is true.

<div v-if="todos.length === 0" class="empty-message"> No todos yet! Add one below. </div> <div v-for="todo in todos" :key="todo.id" class="todo-item"> {{ todo.text }} </div>

Explanation:

  • v-if="todos.length === 0" - checks if the todos array is empty
  • If true, shows the "No todos" message
  • If false, the message is not rendered in the DOM
  • todos.length gives us the number of items in the array

Step 4: Add Checkbox to Mark Todos Complete

What we're doing: Add a checkbox for each todo that toggles the completed property.

Key concept: Use v-model to bind the checkbox to the todo's completed property.

<div v-for="todo in todos" :key="todo.id" class="todo-item"> <input type="checkbox" v-model="todo.completed"> <span>{{ todo.text }}</span> </div>

Explanation:

  • v-model="todo.completed" - creates two-way binding
  • When you check the box, todo.completed becomes true
  • When you uncheck, it becomes false
  • Vue automatically updates the UI when this changes!

Step 5: Style Completed Todos with :class Binding

What we're doing: Use :class to conditionally apply CSS classes based on whether a todo is completed.

Key concept: :class (or v-bind:class) dynamically adds/removes CSS classes.

<div v-for="todo in todos" :key="todo.id" class="todo-item" :class="{ completed: todo.completed }"> <input type="checkbox" v-model="todo.completed"> <span>{{ todo.text }}</span> </div>

Explanation:

  • :class="{ completed: todo.completed }" - adds the "completed" class when todo.completed is true
  • The syntax is: { className: condition }
  • If todo.completed is true, the "completed" class is added
  • If false, the class is removed
/* CSS */ .todo-item.completed { background: #d4edda; text-decoration: line-through; color: #6c757d; }

Step 6: Add Input and Button to Create New Todos

What we're doing: Create a form with an input field and button to add new todos.

Key concept: We'll use a method to add new items to the array.

<div> <input type="text" v-model="newTodoText" placeholder="Add a new todo..." @keyup.enter="addTodo"> <button @click="addTodo">Add Todo</button> </div>

Explanation:

  • v-model="newTodoText" - binds input to the data property
  • @click="addTodo" - calls the addTodo method when button is clicked
  • @keyup.enter="addTodo" - also adds todo when Enter key is pressed (nice UX!)

Step 7: Create the addTodo Method

What we're doing: Write a method that creates a new todo and adds it to the array.

methods: { addTodo() { // Check if input is not empty if (this.newTodoText.trim() === '') { return; // Don't add empty todos } // Create new todo object const newTodo = { id: Date.now(), // Use timestamp as unique ID text: this.newTodoText, completed: false }; // Add to todos array this.todos.push(newTodo); // Clear the input field this.newTodoText = ''; } }

Explanation:

  • this.newTodoText.trim() - removes whitespace and checks if empty
  • Date.now() - creates a unique ID using current timestamp
  • this.todos.push(newTodo) - adds the new todo to the array
  • this.newTodoText = '' - clears the input after adding
